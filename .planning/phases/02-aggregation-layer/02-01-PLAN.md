1. In `src/server/data_dragon.rs`, add a public function `normalize_champion_name(input: &str, champions: &[Champion]) -> Option<String>` with this logic:
   - First: exact match on `c.id == input` — return `Some(input.to_string())`
   - Second: case-insensitive match on `c.name.to_lowercase() == input.to_lowercase()` — return `Some(c.id.clone())`
   - Third: fuzzy match by stripping non-alphanumeric chars and lowercasing both sides (`c.id` stripped+lowered vs `input` stripped+lowered) — return `Some(c.id.clone())`
   - None if no match
   - This function is `#[cfg(feature = "ssr")]` since it's only called server-side.

```rust
Effect::new(move |_| {
    let v = value.get();
    if v != filter_text.get_untracked() {
        // Try to find display name for canonical ID
        let display = champions.with_value(|champs| {
            champs.iter().find(|c| c.id == v).map(|c| c.name.clone())
        });
        set_filter_text.set(display.unwrap_or(v));
    }
});
```
