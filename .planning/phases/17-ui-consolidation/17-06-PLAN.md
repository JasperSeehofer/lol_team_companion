    Total cost estimate for ~3 images: ~$0.15-0.30 either way (negligible).

    Also: confirm which surfaces get backgrounds:
    - MANDATORY (UI-SPEC): closed-beta landing (demacia + pandemonium variants).

  </context>
  <options>
    <option id="option-a">
      <name>fal.ai</name>
      <pros>Recommended in RESEARCH; sub-second latency; consistent flat per-image pricing.</pros>
      <cons>Requires API key onboarding (~5 min friction).</cons>
    </option>
    <option id="option-b">
      <name>replicate.com</name>
      <pros>Equally featured; community trust.</pros>
      <cons>~10% pricier per image; pay-per-second can have unpredictable warmups for one-off jobs.</cons>
    </option>
    <option id="option-c">
      <name>Self-host on GPU rental</name>
