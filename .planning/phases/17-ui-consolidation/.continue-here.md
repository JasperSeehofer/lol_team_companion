## Critical Anti-Patterns

| pattern | severity | description |
|---|---|---|
| CWD-drift during worktree merge | blocking | The merge loop must run with `cd /home/jasper/Repositories/lol_team_companion` AND use `git -C $REPO` for every git call. During Wave 1 cleanup, removing a worktree while the bash CWD was inside it caused the merge commits to land on the worktree branch instead of main. Recovery: `git merge --ff-only <commit>` to reattach main. To prevent: never let bash sessions implicitly inherit CWD inside a worktree, and never `git worktree remove` a worktree if you might be inside it. |
| Spawning parallel agents that overlap on `files_modified` | blocking | Wave 2 had three plans (17-03d, 17-04, 17-05) sharing `17-OD-MAP.md`. They were correctly forced sequential. If Wave 4+ ever has overlapping plans the orchestrator must split into sub-waves. |

## Reusable lessons surfaced (worth `/wiki-debrief` later)

Listed in priority order:

1. **CWD-drift recovery via fast-forward** — when worktree merges land on a non-main branch due to CWD ambiguity, `git merge --ff-only <commit>` from main reattaches the work cleanly. The descendants are still in the object store; only the ref needs to move.
2. **AI-asset graceful degradation pattern (17-06)** — token-gradient placeholder + reproducibility log doubling as the regeneration command. Swap-in is one line per asset when the API key arrives.
3. **Cross-repo design-system seeding + bridge document (17-02)** — DESIGN.md + tokens.css live in sibling repo, OD-MAP.md tracks status (pending → generated → ported) in the consuming repo. Reusable for any "design system + consuming app" or "spec + implementation" pair.
4. **Leptos 0.8 sync-context closures cannot await AuthSession (17-01)** — SSR-authoritative session-derived data requires axum middleware injection.
5. **Scoped CI guardrail introduction with `deferred-items.md` (17-01)** — when adding new style/quality gates to a codebase with widespread legacy violations, scope to plan-touched files and track legacy in a deferred-items.md.
6. **Recursion-limit-safe Leptos restyle (17-03c)** — at `#![recursion_limit = "512"]`, only ONE outer wrapper div + class-string swaps are safe; new `view! {}` macros risk pushing past the limit.
7. **`google-webfonts-helper` variant string convention (17-01)** — `regular` not `400`; `italic` not `400italic`.
8. **CI grep guardrails accidentally match comments (17-01)** — phrase comments without the literal pattern, OR structure greps to skip comment lines.
