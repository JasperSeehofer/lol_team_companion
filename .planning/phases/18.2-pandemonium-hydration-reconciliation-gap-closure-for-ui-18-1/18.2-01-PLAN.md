<!-- Short-circuit mutation pattern (apply to each primitive's is_pandemonium arm) -->

The mutation is identical per cycle — at the TOP of the `if is_pandemonium { ... }` arm body,
insert an early return that emits an empty fragment instead of the rich Pandemonium markup:

  return view! { <></> }.into_any();

This forces the Pandemonium branch to emit zero structural markup. If the panic clears after
this mutation, that primitive is implicated (it was emitting markup that diverged SSR vs WASM).
